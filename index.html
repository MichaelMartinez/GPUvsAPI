Token Calculator

Tokens are calculated based on the input and output of the model. For example, if you generate 32k tokens, 3/4 of that is 24k words. Each page averages around 500 words or 0.5k words. So, 24k / 0.5k = 24 x 2 = ~48 pages.

As a rule of thumb, for every input token, there will be an output token. So generated tokens will be calculated against the input and output price. This isn't a perfect calculation, but it's a good starting point.
